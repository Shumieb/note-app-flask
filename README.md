## Notes App Built using Flask


## CRUD operations
#### GET "/notes" - to get all notes
#### GET "/notes/id" - to get a single note by id 
#### POST "/notes" - to add a new note
#### PUT "/notes/id" - to update a single note by id 
#### DELETE "/notes/id" - to delete a single note by id 
#### GET "/notes/priority/priority" - to get notes by priority
